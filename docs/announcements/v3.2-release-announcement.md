# 🚀 ZaloCRM v3.2 — Bùng nổ với Bot-Auto + Lead Scoring + Redesign toàn diện

**Phát hành:** 21/05/2026
**Release URL:** [github.com/locphamnguyen/ZaloCRM/releases/tag/v3.2.0](https://github.com/locphamnguyen/ZaloCRM/releases/tag/v3.2.0)

Sau gần 1 tuần dồn lực cho **Phase 6 (Lead Scoring)** và **Phase 7 (Bot-Auto Automation)**, hôm nay ZaloCRM chính thức ra mắt **v3.2** — bản nâng cấp lớn nhất kể từ v3.0. Đây không chỉ là một release vá lỗi — đây là cú nâng tầm từ một CRM nhắn tin Zalo đơn thuần thành một **growth automation platform** thực thụ cho team sale.

---

## 🤖 Bot-Auto — Tự động hoá kênh Zalo từ A đến Z

Module **Bot-Auto** mới (top-level tab cạnh "Tin nhắn", "Bạn bè", "Khách hàng") gồm 5 thành phần kết hợp như Lego:

| Thành phần | Vai trò |
|------------|---------|
| **🧱 Blocks** | Khối nội dung tái sử dụng — viết 1 lần, dùng cho mọi sequence/broadcast |
| **🔄 Sequences** | Chuỗi tin nhắn tự động theo cron, tự dừng khi khách phản hồi (stop_on_accept) |
| **⚡ Triggers** | Kích hoạt theo sự kiện: cron, scheduled_cron, webhook (order_success), birthday, request_friend Zalo SDK |
| **📢 Broadcasts** | Gửi batch tới một audience cụ thể, có lịch + scheduler riêng |
| **📋 Lists** (Tệp khách hàng) | Import CSV/Excel + column mapping + inline edit + undo delete + 2-axis status |

### Use-case thực tế
- **Auto chúc mừng sinh nhật** khách → trigger `birthday` → block "🎂 Chúc mừng sinh nhật anh/chị {name}, tặng voucher 10%" → gửi 8:00 sáng hàng ngày
- **Webhook order_success** từ shop → sequence cảm ơn → upsell sau 7 ngày → review request sau 30 ngày
- **Import 5000 SĐT** từ Excel → check Zalo → kết bạn tự động với 4242 friend hợp lệ

> Engine xử lý có **block-bound trigger materializer** + **manual_run** để test trước khi schedule, dùng **Zalo SDK thật** chứ không phải simulation.

## 📊 Lead Scoring — Đừng để khách lead bị "lạnh"

Phase 6 mang về **engine chấm điểm lead tự động** + dashboard cảnh báo:

- **Signal detector** bắt 3 nhóm tín hiệu: inbound (khách nhắn), outbound (mình nhắn), meeting (lịch hẹn)
- **Auto-decay** điểm khi không có tương tác → tránh "false hot lead" tồn đọng
- **7 auto tag** sale dùng để phân khúc nhanh: `cold-lead`, `warm-lead`, `hot-lead`, `champion`, `cooling`, `at-risk`, `dormant`
- **Stuck Detection cron** → trang **`/leads/stuck`** liệt kê KH đình trệ theo từng stage
- **ScoreBreakdownModal** — bấm vào điểm sẽ hiện explainability: "Vì sao KH này được 87 điểm?"
- **`/settings/crm/scoring`** — cấu hình signal weights, thresholds, decay rate riêng cho từng tổ chức

→ Sale không còn phải mở Excel chấm điểm tay. Manager có view "Top hot lead tuần này" + "KH sắp lạnh, cần cứu" ngay trên dashboard.

## 🎨 Redesign toàn diện — Smax / Airtable parity

5 màn hình quan trọng được làm lại từ đầu theo **Airtable-design-analysis spec**:

### AppointmentsView
Quick-create dialog ngay từ chat thread, friendly time picker, list layout settings.

### FriendsView
Flat per-pair table + kind tabs (friend / pending / blocked / recommendation), responsive 4-tier grid (HD / FHD / 2K / 4K).

### ZaloAccountsView
Dashboard 2-axis status: lifecycle (5 ô: pending / connecting / connected / qr_pending / disconnected) + system messages stack.

### Settings layout
Nav grouping hoàn toàn mới: **Personal** / **Team** / **CRM** / **Channels** / **Dev** — dễ tìm hơn 10x so với menu phẳng cũ.

### Top-nav responsive
Fix triệt để: menu luôn hiển thị đầy đủ ở mọi viewport (1024px → 4K). Compact dần theo width, không bao giờ ẩn tab "Cài đặt" sau hamburger menu.

## 🛠 Cách nâng cấp

```bash
# 1. Backup DB (đề phòng cần rollback)
docker exec zalo-crm-db pg_dump -U crmuser zalocrm > backup-v3.1-$(date +%Y%m%d-%H%M).sql

# 2. Pull + rebuild
git pull origin main
docker compose up -d --build app

# 3. Verify
docker logs zalo-crm-app --tail 30 | grep -E "broadcast-scheduler|list-enrichment|cron-scheduler"
```

Container entrypoint tự chạy `prisma db push --accept-data-loss` để sync schema mới (chỉ thêm bảng/field, không drop gì).

Xem hướng dẫn chi tiết + rollback procedure tại [README.md → Nâng cấp từ v3.1 lên v3.2](https://github.com/locphamnguyen/ZaloCRM/blob/v3.2.0/README.md#nâng-cấp-từ-v31-lên-v32).
