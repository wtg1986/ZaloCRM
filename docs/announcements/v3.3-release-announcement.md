## Facebook Lead Ingestion

Module Facebook mới nằm trong **Cài đặt → Channels → Facebook**:

- Kết nối Meta OAuth, chọn page, lưu token page an toàn trong backend.
- Webhook verify + HMAC signature cho lead events.
- Queue xử lý lead bất đồng bộ để không mất lead khi Meta gọi webhook dồn dập.
- Tự khám phá form, tự tạo **Customer List** khoá tên theo page/form.
- Lead mới được tạo Contact/Customer List item và có thể gán sale vòng tròn.
- Cron refresh token để giảm rủi ro page mất kết nối.

Luồng này biến Facebook Lead Ads thành nguồn khách hàng vào CRM, không cần import Excel thủ công.

## Chat media hoàn thiện hơn

- Chuyển tiếp được **hình ảnh, video, audio** thay vì chỉ text.
- Khi forward media, backend tải file từ MinIO/R2/S3 hoặc Zalo CDN URL rồi gửi lại qua Zalo.
- Tin nhắn đến có ảnh/video cũ từ Zalo CDN có thể mirror/backfill sang object storage.
- Video gửi đi có thumbnail để bubble hiển thị đúng preview.
- Kéo thả file/hình/video vào màn hình chat được khôi phục.

## Object Storage: MinIO, S3, Cloudflare R2

v3.3 bổ sung cấu hình R2 vào `.env.example` và chuẩn hoá cách đọc `.env`:

- MinIO local vẫn là mặc định cho Docker Compose.
- Amazon S3 dùng cùng bộ biến `S3_*`.
- Cloudflare R2 dùng endpoint dạng `https://<account-id>.r2.cloudflarestorage.com`.
- Parser env đã xử lý trường hợp password/secret có ký tự `#`, tránh bị cắt nhầm như comment.
